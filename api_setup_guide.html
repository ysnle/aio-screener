📊 개요

AIO Screener는 여러 무료 및 유료 API를 통합하여 실시간 주가, 뉴스, 경제 데이터를 제공합니다. 아래 가이드를 따라 각 API를 설정하면 모든 기능을 활용할 수 있습니다.

Finnhub API

실시간 주가 데이터 & 기업 뉴스
무료
1

Finnhub 가입하기

https://finnhub.io 에 접속하여 'Sign Up' 버튼을 클릭합니다.

이메일을 입력하고 비밀번호를 설정한 후 계정을 생성합니다.

2

API Key 발급받기

로그인 후 대시보드에서 'API Tokens' 섹션을 찾습니다.

무료 플랜 (Free) 에서 자동으로 생성된 API Key를 복사합니다.

✓ 무료 플랜: 60 calls/minute, WebSocket 지원, 최대 30개 심볼

3

AIO 스크리너에 입력하기

AIO 스크리너 좌측 메뉴의 ⚙ 설정 → API Keys 에서

'Finnhub API Key' 입력란에 복사한 키를 붙여넣습니다.

저장 버튼을 클릭하면 자동으로 localStorage에 저장됩니다.

4

제공 데이터 확인

  • 실시간 주가 (WebSocket 기반)
  • 기업 뉴스 및 소식
  • 기본 재무 데이터 (P/E, 배당율 등)
  • 기업 프로필 정보

💡 팁: 무료 플랜만으로도 대부분의 개인 투자자 용도에는 충분합니다. 월 60회 이상의 API 호출이 필요하면 유료 플랜 ($50/월) 업그레이드를 고려하세요.

NewsData.io

글로벌 뉴스 및 시장 뉴스
무료
1

NewsData.io 가입하기

https://newsdata.io 에 접속하여 우측 상단의 'Sign Up' 버튼을 클릭합니다.

이메일 또는 Google/GitHub 계정으로 가입할 수 있습니다.

2

API Key 발급받기

로그인 후 'Dashboard' → 'API Keys' 섹션에서

생성된 API Key를 복사합니다. (자동으로 생성되어 있습니다)

✓ 무료 플랜: 200 credits/day, 50+ 언어 지원, 실시간 뉴스

3

AIO 스크리너에 입력하기

AIO 스크리너의 ⚙ 설정 → API Keys 에서

'NewsData.io API Key' 입력란에 API Key를 붙여넣습니다.

저장 버튼을 클릭합니다.

4

제공 데이터 확인

  • 글로벌 비즈니스 뉴스
  • 기술 및 금융 뉴스
  • 경제 뉴스 (다양한 국가)
  • CORS 지원 (별도 프록시 불필요)

📌 주의: NewsData.io는 CORS (Cross-Origin Resource Sharing) 를 지원하므로, Cloudflare Workers 같은 프록시 없이도 브라우저에서 직접 호출 가능합니다.

Cloudflare Workers

CORS 프록시 서버
무료
1

Cloudflare 계정 가입하기

https://dash.cloudflare.com 에 접속합니다.

'Sign Up' 버튼을 클릭하고 이메일과 비밀번호로 계정을 생성합니다.

2

Worker 생성하기

좌측 메뉴에서 'Workers & Pages' 를 선택합니다.

'Create Worker' 버튼을 클릭합니다.

이름을 입력 (예: "aio-proxy") 하고 'Deploy' 을 클릭합니다.

3

프록시 코드 추가하기

Worker 에디터 창에 다음 코드를 붙여넣습니다:

addEventListener('fetch', event => { event.respondWith(handleRequest(event.request)); }); async function handleRequest(request) { const url = new URL(request.url); const targetUrl = url.searchParams.get('url'); if (!targetUrl) { return new Response('Missing ?url= parameter', { status: 400 }); } try { const response = await fetch(targetUrl, { headers: { 'User-Agent': 'Mozilla/5.0 (compatible; AIO-Screener/1.0)', }, }); const body = await response.text(); return new Response(body, { status: response.status, headers: { 'Access-Control-Allow-Origin': '*', 'Access-Control-Allow-Methods': 'GET, POST, OPTIONS', 'Access-Control-Allow-Headers': '*', 'Content-Type': response.headers.get('Content-Type') || 'text/plain', }, }); } catch (err) { return new Response('Proxy error: ' + err.message, { status: 500 }); } }
4

배포 및 URL 확인하기

코드 우측 상단의 'Save and Deploy' 버튼을 클릭합니다.

배포 완료 후 표시되는 Worker URL을 복사합니다 (예: https://aio-proxy.yourname.workers.dev).

5

AIO 스크리너에 입력하기

AIO 스크리너의 ⚙ 설정 → API Keys 에서

'Cloudflare Proxy URL' 입력란에 Worker URL을 붙여넣습니다.

저장 버튼을 클릭합니다.

✓ 무료 플랜: 100,000 requests/day (충분한 한도)

🔧 사용 방법: CORS 문제가 있는 API 호출 시, https://aio-proxy.workers.dev?url=TARGET_URL 형식으로 요청하면 프록시를 통해 전달됩니다.

Claude API (Anthropic)

시장 분석 AI 챗봇
유료 (저비용)
1

Anthropic 계정 가입하기

https://console.anthropic.com 에 접속합니다.

'Get Started' 또는 'Sign Up' 버튼을 클릭합니다.

이메일로 가입하거나 Google 계정을 사용합니다.

2

API Key 생성하기

로그인 후 좌측 메뉴에서 'API Keys' 를 선택합니다.

'Create Key' 버튼을 클릭하고 이름을 입력합니다 (예: "AIO Screener").

생성된 API Key를 안전한 곳에 복사해둡니다.

⚠️ 이 Key는 다시 보여지지 않으므로 반드시 안전하게 보관하세요!

3

결제 정보 추가하기 (선택사항)

'Billing' 탭에서 신용카드 정보를 추가합니다.

월 한도를 설정할 수 있습니다 (예: $10 한도 설정).

💡 첫 사용 시 $5 크레딧을 제공합니다.

4

AIO 스크리너에 입력하기

AIO 스크리너의 ⚙ 설정 → API Keys 에서

'Claude API Key' 입력란에 API Key를 붙여넣습니다.

저장 버튼을 클릭합니다.

5

모델 선택하기

권장 모델: claude-3-haiku (가장 저렴, 스트리밍 지원)

가격 (2024년 기준):

  • Input: ~$0.25 / 1M tokens
  • Output: ~$1.25 / 1M tokens

✓ 월 예상 비용 (일 50회 질문): $2-5/월

예시: 3,000 토큰 입력 + 500 토큰 출력 = 약 $0.001 (0.1센트)

기타 내장 무료 API

별도 설정 불필요
무료

다음 API들은 AIO 스크리너에 이미 내장되어 있으며, 별도의 API Key 설정이 필요하지 않습니다:

CoinGecko

암호화폐 데이터

  • 실시간 암호화폐 가격
  • 시가총액, 거래량
  • 완전 무료

Exchange Rate API

실시간 환율

  • 150+ 국가 환율
  • 실시간 업데이트
  • 완전 무료

FRED (Federal Reserve)

미국 경제 데이터

  • 기준금리, 실업률
  • GDP, 인플레이션
  • API Key 필요 (무료)

Alpha Vantage

주식 데이터

  • 5분봉, 일봉, 주봉
  • 기술적 지표 (SMA, EMA)
  • 무료: 5 calls/min

Twelve Data

글로벌 금융 데이터

  • 주식, 선물, 암호화폐
  • 다양한 시간봉
  • 무료: 8 calls/min

Financial Modeling Prep

재무제표 데이터

  • 재무제표, 캐시플로우
  • 재무비율
  • 무료: 250 calls/day

RSS2JSON

RSS 피드 변환

  • 뉴스 피드
  • 블로그 업데이트
  • 무료: 10K/day

⚙️ AIO 스크리너 내 설정 방법

단계별 설정 가이드

1
AIO 스크리너를 실행하고 좌측 메뉴의 ⚙ 아이콘을 클릭합니다.
2
'API Keys' 또는 'Settings' 탭으로 이동합니다.
3
각 API에 대해 위 가이드에서 발급받은 API Key를 해당 입력란에 붙여넣습니다.
4
'Save' 또는 '저장' 버튼을 클릭하면 자동으로 브라우저의 localStorage에 저장됩니다.
5
페이지를 새로고침해도 설정이 유지됩니다.

🔒 보안 주의사항:

  • API Key는 브라우저의 localStorage에만 저장됩니다.
  • GitHub Pages에 배포해도 개인 Key는 서버에 저장되지 않습니다.
  • 다른 사람의 컴퓨터에서 사용할 때는 꼭 로그아웃하세요.
  • 프라이빗 브라우징 모드에서는 설정이 저장되지 않습니다.

✓ 설정 완료 체크리스트

  • Finnhub API Key 입력 및 저장
  • NewsData.io API Key 입력 및 저장
  • Cloudflare Worker URL 입력 및 저장
  • Claude API Key 입력 및 저장 (선택사항)
  • 각 API의 실시간 데이터 확인
